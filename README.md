# ⏰ Clock Nano

A simple Arduino-based digital clock project built for learning and experimenting with microcontrollers.

---

## 🔧 Features
- Real-time clock display
- Arduino Nano compatible
- Easy to modify and expand
- Lightweight and fast

---

## 🧠 Project Goal
This project is part of my learning journey with Arduino.  
The goal is to build a stable, upgradeable clock system and improve it over time.

---

## 📁 Project Structure
- `v1-basic/` → initial working version  
- `v2-fix/` → bug fixes and improvements  
- `v3-upgrade/` → added features (WiFi / sensors / etc.)

---

## 🚀 Future Ideas
- WiFi time sync (NTP)
- OLED display upgrade
- Battery + solar power version
- Alarm / timer functions
